## ✅ Resolved

- [x] **Hatchling/nestling phase** — Cut. Run begins at fledgling.
- [x] **Switch target** — Deferred to post-launch port.
- [x] **Starter species** — American Robin only. Not a choice — a beginning. Jon Young / *What the Robin Knows* as explicit touchstone.
- [x] **Egg/chick count** — Species-accurate ranges, capped at 3.
- [x] **Mate killed during breeding** — Second mate possible. Rescue mechanic, not mulligan.
- [x] **Run clock** — No clock. Purely environmental. First Winter death intentional and not softened.
- [x] **Death card design** — Dark Souls x Sanrio. Predation = "{SPECIES} LUNCH". Starvation = "VULTURE FOOD". Human first death: silence only.
- [x] **Season title cards** — Bloom + saturation spike + glowing season name. Opposite visual language from death card. Checkpoint system: quit-and-resume only, death permanent. Saves accumulated landscape state (overharvest, NPC hostility).
- [x] **Controls** — Four buttons, anatomical: A=Feet, B=Beak, X=Wings, Y=Social. Threat suppresses Y entirely. Full contextual priority hierarchy defined.
- [x] **Corvid tool use** — 8 objects fully specced. Rhythm game mechanic. Scarecrow = false threat. Shiny object = 90% useless, 10% [SHINY OBJECT].
- [x] **Corvid NPC recruitment** — Tap Y (caw). Dumpster requires ≥1 NPC. More crows = easier sequence.
- [x] **Human type spectrum** — Tourist, Indigenous hunter (ambiguous), trophy hunter (lethal), poisoner (silent lethal), farmer (seasonal).
- [x] **Farmer ecology** — Full seasonal system. Spray = silent poison in Summer.
- [x] **World scale system** — 5 tiers (Tiny 1× → Very Large 30×). Dynamic streaming required at Large+. Flagged.
- [x] **Threat viability matrix** — Predators viable at ecological size tiers only. Human threats viable at all tiers.
- [x] **Collective vs. solitary awareness** — Gregarious: flock geometry determines speed/bounds of awareness. Solitary: sharper baseline, higher stillness ceiling.
- [x] **Parids class** — 12th V1 class. Chickadee as flock leader. Player alarm calls propagate to all NPC birds in range.
- [x] **Nest defense** — Fledgling meters, heartbeat score layer, nest-anchored camera. Each fledge = musical swell.
- [x] **Unlock tree** — Full DAG, Robin at root. 12 classes, all species, Bronze→Platinum. Warblers ungated beyond Yellow Warbler.
- [x] **Heron unlock** — Via Gull tree. Pelican is the bridge.
- [x] **Mimid unlock** — Via successful mate attraction ×3.
